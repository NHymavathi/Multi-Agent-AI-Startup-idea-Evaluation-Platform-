# AI Startup Idea Validator

A premium multi-agent startup evaluation web app built with Flask, vanilla HTML/CSS/JavaScript, and Groq AI.

## Run locally

1. Install dependencies:
   pip install -r requirements.txt
2. Add your Groq API key to the .env file.
3. Start the app:
   python app.py
4. Open http://localhost:5000

The app works with local fallback logic even without an API key, and it will use Groq when configured.
